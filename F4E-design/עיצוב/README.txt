כמו שתוכלו לראות, בתקייה הזאת נמצאים תמונות של כל המסכים שעיצבתי. 

בכל אחד מהתמונות האלה, סימנתי שלושה דברים:

	א. המספרים שנמצאים בתוך עיגולים מסמלים את תמונות הPNG (שבתקיית PNG).
	ב. בתוך הריבועים השחורים כתבתי את ההערות שלי.
	ג. השתמשתי בפונט Assistant (מוזמנים להוריד אותו מכאן: https://fonts.google.com/specimen/Assistant). 
	   בתוך ריבועים לבנים שקופים כתבתי את גודל הטקסט ואת המשקל שלו:
		R = Regular 
		SB = Semi-bold
		B = Bold
		XB = Extra-bold
	  כך, למשל, "XB, 18" אומר גודל גופן 18 במשקל Extra-bold, וכן הלאה...

וכמובן, אם אתם צריכים עזרה או הסבר נוסף, אל תהססו לפנות אליי!
